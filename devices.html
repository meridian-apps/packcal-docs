If your watch isn't here, it's most likely a pre-Connect IQ 5.0 model. The Connect IQ Store listing has the live compatibility check, install only completes on a supported device.

One field, every screen size

A data field's cell can be anything from a full single-field screen on a Forerunner 965 down to a quarter of the screen on a four-field Instinct. PackCal draws by hand for the cell it's given: on a spacious single-field screen it shows the calorie hero plus a live rail, the pack-bonus pill, and a footer; in a denser multi-field layout it degrades to a labelled number, then to a bare number when the cell is tiny. The layout is also display-shape-aware (round and rectangular watches each get a layout tuned to their bezel), with a condensed, sun-legible numeral face.

Connect IQ version

PackCal builds against Connect IQ API 5.0. Most listed devices have firmware that supports newer Connect IQ features; a few older devices fall back to a 5.0-compatible code path automatically. You don't need to pick, the right path runs on the right device.
